Analyzed Weather and Hotels to Match

Everyone complains about the weather, but nobody ever seems to do anything about it. - Willard Scott

Throw a dart and pack your bag

Where would you land on a random pick? This project eliminates the risky business of blindfolds and pointy objects by utilizing randomized geocoordinates based on likely weather outcomes as provided by the OpenWeatherMap and Geopify API developer tools and some hemisphere specific analysis.

API pulls and analysis leveraged with Python. Interactive map built using D3.js and Leaflet. Code repository: Weather Analysis Redux
